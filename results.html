Prediction System Results

Learn about the results of our machine learning testing and evaluation process.

Model Selection

We tested various machine learning models to identify the best performing ones for predicting NBA game outcomes:

  • Logistic Regression (LR)
  • K-Nearest Neighbors (KNN)
  • Random Forest (RF)
  • Gaussian Naive-Bayes (GNB)
  • Support Vector Machines (SVC)
  • Neural Network (NN)
  • Elo Rating System

Results

Top Performing Models

The models that achieved the highest accuracy were:

  • Elo Rating System: 65.3% accuracy
  • Gaussian Naive-Bayes (GNB): 63.5% accuracy
  • Random Forest Classifier (RFC): 63.2% accuracy

Dataset Insights

The top-performing dataset was the 20-game Four Factors dataset, which provided consistent and detailed input data for training and evaluation.

Model Comparison

Accuracy across the tested models ranged between 57% to 65%. Interestingly, error rates were higher during the first half of the season, likely due to team adjustments and incomplete data.

Feature Selection

High feature counts led to increased accuracy despite collinearity among some stats:

  • Shooting and scoring stats showed significant correlations.
  • Rebounding stats required careful handling due to overlap with other metrics.

Next Steps

While the project successfully identified promising models, it fell short of the original accuracy goal of 68%. Future improvements will focus on:

  • Extensive data collection for broader training datasets.
  • Player-specific aggregated data for deeper insights.
  • Further adjustments to model parameters and feature selection.
